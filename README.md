## How to Run
Run the following directly under the project.
```
$ go run ./ex[nn]
```
### Example
```
$ go run ./ex07
What is the length of the room in feet? 15
What is the width of the room in feet? 20
You entered dimensions of 15 feet by 20 feet.
The area is
300 square feet
27.871 square meters
```
